# IF2000_Proyecto_Final

Descripción del proyecto
Sistema básico de reservación de tiquetes aéreos desarrollado en Java con programación orientada a objetos.
Permite:

Seleccionar vuelos según origen y destino.

Elegir clase (ejecutiva o económica).

Verificar disponibilidad de asientos (2 ejecutiva y 2 económica por avión).

Generar boleto y factura con la información del pasajero y del vuelo.

El proyecto está pensado como una simulación académica sin conexión a base de datos, con interfaz gráfica.

Objetivo:
Aplicar conceptos de POO y prácticas de control de versiones en GitHub a través de un sistema práctico y funcional.
